Managing Signed Keys with Machine Owner Key

Sun 23 July 2017
By 陳鐸元

Introduction

With MOK we can sign, manage, and delete keys for installed kernel modules.

Usage

Enrolling a key

The guide can be found here.

List all the enrolled keys:

`mokutil -l`

Deleting a key

First, you have to export the keys: mokutil --export

Delete the key: mokutil -dTheFile \t E.g. mokutil -d MOK-0001.der

Proudly powered by Pelican, which takes great advantage of Python.
